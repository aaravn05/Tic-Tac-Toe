# Tic Tac Toe (C++)

A terminal-based Tic Tac Toe game written in C++.

## Features

- Human vs Computer
- Random AI
- Win detection
- Tie detection
- Input validation
- Modular functions

## Concepts Practiced

- Functions
- Arrays
- Loops
- Input validation
- Game state management
- Random number generation
